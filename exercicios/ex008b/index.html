Pré-formatação

Usando a tag PRE é possível deixar o código do jeito que está escríto no arquivo HTML. Por exemplo:

num = int(input('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Nota-se que o código não foi quebrado, mesmo usando a tag CODE

Agora iremos utilizar a tag PRE no código

            
                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print(f'O número {num} é PAR')
                else:
                    print(f'O número {num} é ÍMPAR')
                print('Fim do programa')

Veja que ficou um espaço ao lado do código. Caso queira eliminar esse espaço, vá no seu arquivo HTML, selecione o código (apenas ele, sem as tags) e pressione Shift+Tab até ele ter ido todo para o canto.

            
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa')

Caso queira fazer o contrário, selecione e utilize apenas a tecla Tab
