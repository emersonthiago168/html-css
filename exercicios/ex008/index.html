Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica. Apenas tem forma).

Nesta frase temos um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica)

Nessa frase temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag MARK

E no outro parágrafo, temos outro texto marcado no final usando a tag MARK

Texto grande e pequeno

Nesse parágrafo temos um texto grande usando a tag BIG que está em desuso

Nesse parágrafo temos um texto pequeno usando a tag SMALL

ATENÇÃO

A tag BIG está em desuso porque sua única função é aumentar o texto visualmente, algo que deve ser feito com CSS. Já a SMALL tem todo um sentido por trás, por exemplo: ''Leia as letras pequenas (SMALL) do contrato''

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag INS

ATENÇÃO

Existe uma tag U para sublinar com o visual idêntico à tag INS, porém se encontra em DESUSO atualmente

Texto sobrescrito

Podemos escrever coisas do tipo x20+3 usando a tag SUP

Texto subscrito

Podemos escrever coisas do tipo h2O usando a tag SUB
